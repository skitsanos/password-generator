# Password Generator

A fast, reliable password generator built for everyday work. Generate strong, random passwords in seconds with a clean interface and mobile‑friendly copy, so you can move on with confidence.

## Why This
- Instant, secure password generation with modern UX
- Simple controls that match how people actually work
- Mobile‑first copy experience that just works

## Features
- Adjustable length (8–32)
- Letters + optional numbers and symbols
- One‑click copy to clipboard with mobile fallback
- Secure randomness via Web Crypto

## Use Cases
- Quick account setup and resets
- Admin workflows and ops tasks
- Everyday personal security hygiene

## Tech Stack
- React + Umi
- Arco Design
- Web Crypto API

## Getting Started
Install dependencies:
```
bun install
```

Run locally:
```
bun run dev
```

Build for production:
```
bun run build
```

## Deployment
```
bun run deploy
```
